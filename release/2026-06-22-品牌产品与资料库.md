# Release Notes - 2026-06-22

## 变更范围

本次 release 聚焦“品牌 / 产品与服务”模块，完善企业多品牌、产品与服务、竞品关系和多模态资料管理能力。

- 品牌档案
- 产品 / 服务档案
- 竞品配置
- 品牌事实 / 资料库
- Query 集关联对象

## 品牌档案

### 多品牌管理

新增和优化：

- 品牌档案调整为当前企业下的品牌列表。
- 支持一个企业维护多个自有品牌。
- 自有品牌与外部竞品品牌分开管理。
- 支持品牌名称、英文名和别名搜索。
- 支持行业、状态筛选。
- 支持新增、编辑、启用和停用品牌。
- 支持分页。
- 支持展示产品 / 服务、竞品和 Query 集关联数量。

### 品牌基础档案

创建和编辑品牌时支持维护：

- 品牌 Logo
- 品牌名称、英文名称
- 法定主体名称
- 品牌简介
- 品牌定位
- 所属行业
- 主要市场 / 地区
- 成立时间、总部地区
- 品牌口号
- 核心人群
- 价格区间
- 品牌官网
- 品牌别名
- 排除词 / 歧义说明
- 负责人

品牌 Logo 支持图片选择和即时预览，关键基础字段创建时必填。

## 产品 / 服务档案

### 对象模型调整

新增和优化：

- “产品档案”调整为“产品 / 服务档案”。
- 支持产品和服务两种档案类型。
- 解决方案、套餐等通过品类 / 服务类型表达。
- 产品与服务共享名称、品牌、定位和识别边界等公共信息。
- 产品和服务使用不同的专属字段。
- Query 集底层关联字段由 `product` 调整为 `offering`。

### 产品专属信息

产品档案支持：

- 型号 / 产品代码
- 价格区间
- 主要使用场景
- 上线时间
- 在售、预售、已下线状态

### 服务专属信息

服务档案支持：

- 交付方式
- 结算方式
- 服务区域
- 服务承诺 / SLA
- 收费方式 / 价格
- 服务中、已下线状态

### 列表管理

- 支持名称、英文名和别名搜索。
- 支持品牌、类型和状态筛选。
- 支持新增、编辑、启用和下线。
- 支持分页。
- 表格简化为名称、品牌、类型、价格 / 收费、状态、更新时间和操作。
- 详细定位、简介和关联统计不再堆叠在列表中。

## 竞品配置

### 竞品关系

竞品配置从竞品卡片调整为竞品关系列表。

支持的竞争层级：

- 品牌
- 产品
- 服务

支持的关系类型：

- 直接竞品
- 间接竞品
- 替代方案
- 标杆对象
- 潜在竞品

新增和优化：

- 支持新增、编辑、启用和停用竞品关系。
- 支持配置我方品牌、产品或服务。
- 支持配置外部竞品对象。
- 支持主要竞争场景、重叠范围、核心人群和监测优先级。
- 支持我方品牌、竞争层级、优先级和状态筛选。
- 支持分页。

### 外部竞品对象

- 外部竞品对象与企业自有品牌档案分开保存。
- 新增竞品关系时可以选择已有外部对象。
- 支持在关系弹窗内创建新的外部品牌、产品或服务。
- 外部对象创建完成后自动回填到当前竞品关系。
- 外部对象支持名称、所属品牌、品类、官网、别名和简介。

## 品牌事实 / 资料库

### 资料形态

“品牌事实 / 知识资料”调整为“品牌事实 / 资料库”。

资料库内部拆分为两个子页面：

- 文本资料
- 图片集

不新增左侧菜单，通过资料库页面内部 Tab 切换。

### 文本资料

新增和优化：

- 支持 Markdown 格式存储。
- 支持 Markdown 编辑和实时预览。
- 支持关联品牌和产品 / 服务。
- 支持业务类型、来源、状态和负责人管理。
- 支持草稿、已生效和已过期状态。

### 图片集

图片不再作为单条顶层资料保存，调整为图片集模型：

```text
图片集
├── 图片 1
├── 图片 2
└── 图片 N
```

新增和优化：

- 一个图片集支持包含多张图片。
- 支持批量上传图片。
- 图片集支持名称、说明、关联品牌和产品 / 服务。
- 每张图片独立维护：
  - 图片标题
  - 图片说明
  - Alt 文本
  - OCR 识别文字
  - 文件格式和大小
- 列表展示图片集包含的图片数量。

### 业务类型

内容形态与业务类型分开管理。

支持的业务类型：

- 品牌事实
- 产品 / 服务事实
- 政策与权益
- 官方口径
- FAQ
- 案例与荣誉
- 媒体素材
- 过期说法

### 链接导入

新增链接导入交互：

- 支持输入网页链接。
- 网页正文转换为 Markdown。
- 支持编辑转换后的 Markdown。
- 支持选择需要保留的网页图片。
- 无关图片可以取消选择。
- 确认入库后生成：
  - 1 条文本资料
  - 1 个包含所选图片的图片集
- 未选择图片时只生成文本资料，不创建空图片集。
- 文本资料和图片集保留原始 URL 和来源网站。

当前版本完成链接导入的 Mock 交互流程，真实网页抓取、正文抽取、图片转存和 OCR 需要后端任务服务接入。

## Query 集关联调整

新增和优化：

- “关联产品”统一调整为“关联产品 / 服务”。
- Query 集支持关联具体产品或服务。
- Query 集列表、详情和 AI 推荐弹窗统一使用产品 / 服务表达。

## 通用交互

本次统一的页面规范：

- 管理列表优先展示可快速扫描的信息。
- 详细档案内容放入新增、编辑或后续详情页面。
- 列表操作统一使用图标按钮。
- 状态徽标禁止换行。
- 信息密集表格设置稳定最小宽度。
- 新增和编辑完成后使用 Toast 提示结果。
